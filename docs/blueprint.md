# **App Name**: Symbolic Care Hub

## Core Features:

- Home Page Banner: Display a hero image with the text "A Reliable and Ethical GP led Care Provider" using the image provided by user.
- Service Directory: Display the directory of services provided by the Adult Health Care Center each one clickable with its details, like live support, in-home care, personal emergency response, etc. to allow customers to easily navigate all the different service lines provided by Symbolic Care
- Contact Form: Create a Contact Form in the contact us page
- Standard Pages: Implement the basic layout and content structure for all the required pages like Home, About Us, Why Choose Us, Services, Trainings, Careers, Support, Contact Us.
- Header and Footer: Create site wide headers and footers

## Style Guidelines:

- Primary color: Light blue (#ADD8E6) to reflect the logo's color scheme and evoke a sense of calmness and trust. It harmonizes well with healthcare-related imagery and design.
- Background color: Very light desaturated blue (#F0F8FF) for a clean, calming backdrop that doesnâ€™t distract from the content.
- Accent color: Slightly darker blue (#87CEEB) to complement the primary color while adding visual interest and highlighting key interactive elements, like links and button hovers.
- Font: 'PT Sans', a humanist sans-serif for headers and body to combine a modern look with warmth.
- Use clear, recognizable icons relevant to the services provided, in a style matching the overall blue color scheme.
- Ensure a clean and responsive design with clear content sections and easy navigation.
